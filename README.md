Данный проект создан для автоматического API тестирования сайта-заметницы

Все зависимости для работы проекта описаны в файле `requirements.txt`, и их необходимо установить в среду командой
```bash
pip install -r requirements.txt
```
Перед запуском тестов необходимо создать файл `.env` с данными учётных записей, пример описан в файле `.env.example`.

Тесты содержаться в папке `tests`, могут запускаться как через среду разработки, как и через команду
```bash
pytest tests/ -v
```

В папке `api` содержатся функции и классф для взаимодействия с api.
В папке `assertions` находятся классы проверок для тестов.
В папке `models` модели pydantic для проверки схем ответа.
В папке `utilities` вспомогательные классы-утилиты (для работы с json и.т.д).
